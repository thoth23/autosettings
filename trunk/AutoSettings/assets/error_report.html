You can use this screen to report an error by email to the author of Timeriffic.

When you select the button below, an email will be generated that contains:

  • The last Java exceptions (a.k.a. 'force close').
  • The last settings changes performed.
  • A summary of your profiles.
  • The device model and build number.
No personal identification is reported.
The mail will open in your default Mail application where you get a chance to review it before sending it.
