Usted puede utilizar esta pantalla para informar de un error por correo electrónico al autor de Timeriffic.

Cuando se selecciona el botón de abajo, un correo electrónico será generado que contiene:

  • Las última excepciones de Java (los mensajes 'forzar cierre').
  • Los último cambios que hizo Timeriffic.
  • Un resumen de sus perfiles.
  • El modelo de teléfono Android.
Ninguna información personal esta utilizada en el correo electrónico.
El correo electrónico se abrirá en su aplicación predeterminada de correo electrónico donde tiene la oportunidad para revisar y editar antes de enviarlo.
