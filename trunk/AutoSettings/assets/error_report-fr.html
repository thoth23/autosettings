Vous pouvez utiliser cet écran pour reporter une erreur par émail à l'auteur de Timeriffic.

Lorsque vous sélectionnerez le bouton ci-dessous, un émail sera généré avec le contenu suivant:

  • Les dernières exceptions Java (les messages 'forcer la fermeture').
  • Les dernières changements de paramètres effectués par Timeriffic.
  • Un résumé de vos profils.
  • Le modéle de votre télephone Android.
Aucune information personnelle n'est utilisée dans l'émail.
L'émail s'ouvrira dans votre application d'émail par défaut où vous aurez la possibilité de voir l'émail et de l'éditer avant de l'envoyer.
